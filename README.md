#  Redmine Help Me Track 

*Runs on Python 2.7* 

### To run the project


```
pip install requests

pip install pyinstaller

python tkinter_main.py

```

### to build, on windows 

```
.\single_build.bat

```

### to build, on ubuntu

```
.\single_build.sh

```


### Work is in progress for the pyside version to run that 

```

pip install pyside

python pyside_main.py

```
